# Introdução a Linguagem Python   

A linguagem Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, criada por Guido van Rossum e lançada pela primeira vez em 1991. Python é conhecida por sua sintaxe clara e legibilidade, o que a torna uma escolha popular tanto para iniciantes quanto para desenvolvedores experientes. A linguagem suporta múltiplos paradigmas de programação, incluindo programação orientada a objetos, funcional e imperativa.

## Características Principais
- **Sintaxe Simples e Legível**: Python utiliza uma sintaxe que enfatiza a legibilidade do código, facilitando a aprendizagem e manutenção.
- **Interpretada**: Python é uma linguagem interpretada, o que significa que o código é executado linha por linha, facilitando o desenvolvimento e a depuração.
- **Tipagem Dinâmica**: Python utiliza tipagem dinâmica, permitindo que os tipos de dados sejam determinados em tempo de execução.
- **Biblioteca Padrão Extensa**: Python possui uma vasta biblioteca padrão que oferece
muitas funcionalidades prontas para uso, desde manipulação de arquivos até operações matemáticas avançadas.
- **Comunidade Ativa**: Python possui uma comunidade global ativa que contribui com pacotes, bibliotecas e frameworks, facilitando o desenvolvimento de aplicações em diversas áreas.   
## Aplicações Comuns
- Desenvolvimento Web (ex: Django, Flask)       
- Análise de Dados e Ciência de Dados (ex: Pandas, NumPy, Matplotlib)
- Inteligência Artificial e Machine Learning (ex: TensorFlow, Scikit-learn)     
- Automação e Scripting
- Desenvolvimento de Software e Prototipagem Rápida     
## Exemplos de Código
```python   
print("Hello, World!")
```

```python   
def swap(a, b):
    return b, a 
x = 5
y = 10  
print("Before swap: x =", x, ", y =", y)
x, y = swap(x, y)
print("After swap: x =", x, ", y =", y)
```
